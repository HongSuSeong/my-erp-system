### Backend
```
- 언어: Kotlin
- 프레임워크: Spring Boot 3.4.10
- ORM: JPA (Hibernate)
- 빌드: Gradle (Kotlin DSL)
- API: RESTful API
```

### Frontend
```
- 프레임워크: Vue 3 (Composition API)
- 상태관리: Pinia
- UI 라이브러리: Element Plus 또는 Vuetify
- 빌드: Vite
```

### Database
```
- Oracle Database 19c (Autonomous Database Free Tier)
- 연결: Oracle JDBC Driver
